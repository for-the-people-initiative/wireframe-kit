# InputIcon

Een icoon gepositioneerd binnen een invoerveld (links of rechts). Puur decoratief (`aria-hidden`).

## Wanneer gebruiken ✅

- **Zoekicoon in zoekveld**: Vergrootglas links
- **Visuele hint**: Kalender-icoon bij datumveld, slot-icoon bij wachtwoord
- **Consistente styling**: Iconen die visueel "in" de input zitten

## Wanneer NIET gebruiken ❌

- **Interactieve iconen** (klik om te wissen, toggle wachtwoord) → gebruik een Button
- **Als enige aanduiding** — iconen zijn decoratief, niet vervanging voor labels
- **Met InputGroupAddon** → kies één van de twee, niet beide

## Alternatieven

| In plaats van InputIcon... | Gebruik... | Wanneer... |
|---|---|---|
| InputGroupAddon | Je een visueel afgescheiden blok wilt (met achtergrond) |
| Button in input | Het icoon klikbaar moet zijn |
| Placeholder tekst | Een tekstuele hint voldoet |

## Best practices

✅ Gebruik `position="left"` voor context-iconen (zoek, kalender)

✅ Gebruik `position="right"` voor status-iconen (valid, loading)

❌ Gebruik niet meer dan 1 icoon per kant

❌ Vertrouw niet op het icoon als enige communicatie — voeg altijd een label of placeholder toe
