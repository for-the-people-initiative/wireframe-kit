# Slider

Een range-input waarmee de gebruiker een numerieke waarde selecteert door te slepen.

## Wanneer gebruiken ✅

- **Bereik instellen**: Prijs, afstand, volume, helderheid
- **Filters**: "Maximaal 50km" of "Budget €200-€800"
- **Instellingen**: Volume, transparantie, grootte

## Wanneer NIET gebruiken ❌

- **Exacte waarden nodig** — een InputText met type number is preciezer
- **Heel groot bereik** (0-10.000) — de slider wordt onnauwkeurig
- **Binaire keuze** (aan/uit) → gebruik [ToggleSwitch](?path=/docs/components-toggleswitch-rationale--docs)
- **Discrete opties** (S/M/L/XL) → gebruik [SelectButton](?path=/docs/components-selectbutton-rationale--docs)

## Alternatieven

| In plaats van Slider... | Gebruik... | Wanneer... |
|---|---|---|
| InputText (number) | Exacte waarde moet worden ingetypt |
| SelectButton | Beperkte discrete opties |
| Select | Veel opties maar exacte selectie |

## Best practices

✅ Toon altijd de huidige waarde naast de slider

✅ Gebruik logische `min`, `max` en `step` waarden

✅ Combineer met een numeriek inputveld voor exacte invoer

❌ Gebruik geen slider voor waarden waar de gebruiker een exact getal moet kiezen

❌ Maak de step niet te klein bij een groot bereik
